void loadFullDatasetFromCsv ( )

Loads the values from a CSV file into the training and data sets according to the training ratio.

Loads the CSV file located in ../datasets/full/DATASET.csv. This is a full dataset, therefore it is necessary to split this data set into test and training data set. This function makes this splitting and saves the values read from the CSV file in the proper matrixes (testSet and trainingSet).

void loadSummariesFromCsv ( )

Loads a CSV file into the summaries (means and stdevs) arrays.

The CSV file is loaded from ../datasets/summaries/DATASET.csv and put into the means and stdevs matrix.

void loadTestsetFromCsv ( )

Loads the test dataset from a CSV file.

The test dataset is loaded from ../datasets/test/DATASET.csv and put into the testset matrix.
